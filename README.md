# Wishflow · 愿航 (Supabase + Next.js)

This project is a bilingual (EN/中文) web app with Google + email/password login.
Default language is English, toggle at the top-right.

## Quick start
```
npm install
npm run dev
```

## Supabase setup
See `supabase/README.md` and apply `supabase/schema.sql`.

## Env
Create `.env.local` from `.env.local.example`.
